# CLAUDE.md - Workspace `D:\Repos`

Arquivo de compatibilidade para fluxos que ainda procuram `CLAUDE.md`.

## Fonte de verdade

- Contrato principal do workspace: `D:\Repos\AGENTS.md`
- Comportamento e limites: `D:\Repos\SOUL.md`
- Contexto do usuario: `D:\Repos\USER.md`
- Atalhos operacionais: `D:\Repos\TOOLS.md`
- Heartbeat: `D:\Repos\HEARTBEAT.md`
- Memoria curada: `D:\Repos\MEMORY.md`
- Memoria diaria: `D:\Repos\memory\`

## Regra pratica

- Se um fluxo ler `CLAUDE.md`, trate este arquivo apenas como ponte.
- O contrato operacional atual vive em `AGENTS.md`.
- Em pedidos de codigo para repos do swarm, seguir o roteamento definido em `AGENTS.md`.
- Antes de alterar codigo em um repo, ler o `AGENTS.md` ou `CLAUDE.md` local do proprio repo, conforme o contrato que aquele repo ainda expuser.

Este arquivo foi reduzido de proposito para evitar drift com `AGENTS.md`.
