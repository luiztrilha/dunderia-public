# Support

## Where To Ask What

- Bugs and regressions: open a GitHub issue
- Feature requests: open a GitHub issue with the use case and constraints
- Security reports: follow [SECURITY.md](SECURITY.md)
- Informal usage questions: Discord, when a conversational back-and-forth is easier

## When Opening An Issue

Please include:

- OS and architecture
- provider in use (`claude-code`, `codex`, `gemini`, `gemini-vertex`, `ollama`)
- how the app was installed (`npm`, source build, local binary)
- exact command used
- relevant logs or screenshots, with secrets redacted

## What Not To Post Publicly

- API keys or tokens
- private chat content unless it is necessary and redacted
- local file paths that contain sensitive information
- exploit details for security issues
